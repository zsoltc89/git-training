  1. Download and install Git from:
    • https://git-scm.com/download/win
    • The default options are ok.
  2. Run "Git Bash" from Start Menu

Forking a repository

  1. Fork this repository:
    • https://github.com/jvalleroy/git-training

Creating a local clone

  1. git clone https://github.com/username/git-training.git
  2. cd git-training

Changing a file

  1. Open index.html in text editor.
  2. Change the title of this presentation.
  3. Save the modified file.

Viewing status

  • git status

Viewing local differences

  • git diff

Staging a change

  1. git add index.html
  2. git status

Viewing local staged differences

  • git diff --staged

Committing a change

  • git commit -m "Change slide text"

Viewing the log

  • git log

Viewing differences to remote

  • git diff origin/git-training

Pushing a change

  • git push

Creating a branch

  • git checkout -b fix-spelling-errors

Pushing a branch

  • git push origin fix-spelling-errors

Creating a pull request on GitHub

Adding a remote repository

  • git remote add upstream https://github.com/hakimel/reveal.js.git

Fetching changes from a remote

  • git fetch upstream

Pulling changes

  • git pull origin git-training

Checking out a remote branch

  • git checkout -b upstream-master upstream/master

Rebasing a branch

  1. git checkout git-training
  2. git rebase upstream-master

Merging a branch

  • git merge fix-spelling-errors

Deleting a branch

  • git branch -D fix-spelling-errors

Merging a pull request

  1. git checkout master
  2. git pull origin master
  3. git remote add contributor-name https://github.com/contributor-name/reveal.js.git
  4. git fetch contributor-name
  5. git merge feature-branch
  6. git push

Creating and viewing tags

  • git tag release-1
  • git tag
